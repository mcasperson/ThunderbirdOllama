This is the email address that summary emails will be sent to.



This is an optional alias added to the email address to make it easier to filter summary emails.

For example, if the email address is test@example.org, and the alias is summary, summary emails are sent to test+summary@example.org.



This is the model used to summarize the email. These models must be pulled by Ollama with the command "ollama pull <modelname>".



This is the size of the context window. Larger sizes allow larger emails to be summarized, but consume more memory and take longer.



The custom instructions to be used for summarization.
